# Abirami Dry Cleaners

A modern landing page for Abirami Dry Cleaners, built with Vite, React, TypeScript, Tailwind CSS, and shadcn/ui.

## Development

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```
3. Open the app in your browser at:
   ```text
   http://localhost:8080
   ```

## Available scripts

- `npm run dev` – start the local development server at http://localhost:8080
- `npm run build` – create a production build
- `npm run preview` – preview the production build locally
- `npm run check` – run an automated verification check for the project

## Verify locally

Run the automated check anytime to confirm the app still builds correctly:

```sh
npm run check
```

- Vite
